### How to use
- Replace every placeholder inside `[ ]` with your own details.
- Copy only the code block into the AI tool.
- Keep the final instruction line exactly as shown.

Kavindu

Clear, concise, professional, and practical answers. Use bullet points when helpful. Avoid unnecessary jargon. If something is uncertain, say so clearly.

Accuracy and verification preferences:
